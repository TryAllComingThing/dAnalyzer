# 报告样例

## 周报样例

```
# 电商业务 周报
## 2026年第17周 (4/21 - 4/27)

### 核心指标概览

| 指标 | 本周 | 环比 | 同比 |
|------|------|------|------|
| 销售额 | 128.5万 | +5.2% | +15.8% |
| 订单量 | 8,230 | +3.1% | +12.4% |
| 客单价 | 156元 | +2.0% | +3.0% |
| 活跃用户 | 15,680 | +8.5% | +22.1% |

### 核心发现

1. 销售额环比增长5.2%，主要来自周末促销活动
2. 活跃用户增长明显，环比+8.5%
3. 转化率略有下降，需要关注

### 异常分析

- 转化率下降至3.2%（上周3.5%），可能是流量质量问题
- 建议：排查流量来源，优化落地页

### 下周建议

1. 继续推广周末促销模式
2. 优化落地页提升转化率
3. 关注退货率异常类目
```

```
# 数据质量报告
## 2026-04-27

### 校验结果汇总

| 校验项 | 状态 | 详情 |
|--------|------|------|
| 空值校验 | 通过 | 空值占比<1% |
| 异常值校验 | 预警 | 销售额有3条异常 |
| 重复值校验 | 通过 | 无重复记录 |
| 断层校验 | 通过 | 日期连续 |

### 问题清单

1. 销售额字段存在3条异常值（单日>50万）
2. 部分商品类目缺失

### 处理建议

- 异常值已标记，建议人工确认
- 缺失类目已用"未知"填充
```

## RFM分析报告样例

```
# 用户RFM分析报告
## 2026年4月

### 用户分层统计

| 分层 | 用户数 | 占比 | 消费金额 |
|------|--------|------|----------|
| 高价值用户 | 1,250 | 8% | 45.2万 |
| 潜力用户 | 3,780 | 24% | 38.5万 |
| 一般用户 | 6,250 | 40% | 28.8万 |
| 流失风险 | 2,500 | 16% | 8.2万 |
| 休眠用户 | 2,220 | 12% | 3.8万 |

### 运营建议

- 高价值用户：提供VIP服务，专属优惠
- 潜力用户：提升消费频次，发放优惠券
- 流失风险用户：实施召回计划
- 休眠用户：唤醒活动，大额优惠
```
