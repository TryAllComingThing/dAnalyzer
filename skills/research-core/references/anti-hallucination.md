# 抗幻觉协议

以下规则**绝对不可违反**。编造事实 = 报告作废。

## 来源归属格式

```
✅ "根据 [1]，2025 年全球市场规模达到 240 亿美元。"
✅ "Smith 等人 (2024) 发现治疗组死亡率降低 23% (p<0.01) [3]。"
✅ "[5] 报告了中国市场的三项主要变化：..."

❌ "研究表明市场规模持续增长。"           ← 哪个研究？
❌ "专家认为这一趋势不可逆转。"            ← 哪些专家？
❌ "该机制是通过氧化应激介导的。"          ← 没有引用，冒充事实
❌ "多个来源证实了这一发现。"              ← 哪些来源？
```

## 事实 vs 综合 vs 推测

```
✅ 事实（有来源）："根据 [2]，2024 年 Q3 用户增长率为 12%。"
✅ 分析（你推的）："这表明市场可能正在加速渗透..."
✅ 推测（明确标注）："如果该趋势持续，预计 2026 年将达到..."

❌ 推测冒充事实："市场规模为 240 亿美元。"（无来源）
❌ 模糊归属："近年来该领域发展迅速。"（无数据、无时间、无来源）
```

## 不确定时

```
✅ "未找到直接涉及 X 的公开来源。"
✅ "关于 Y 的结论基于单一来源 [7]，建议交叉验证。"
❌ 编造引用来填补空白
❌ 用 plausible-sounding 的概括掩盖信息缺失
```

## 高风险场景识别

具体数字（金额/比例/日期）、人名/公司/产品名、因果关系陈述、预测性结论——这些**必须**有明确来源，**必须**交叉验证。
