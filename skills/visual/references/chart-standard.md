# 可视化规范 (Chart Standards)

## 图表选择指南

| 数据特征 | 推荐图表 |
|----------|----------|
| 时间趋势 | 折线图、面积图 |
| 分类对比 | 柱状图、条形图 |
| 部分占比 | 饼图、环形图 |
| 分布特征 | 直方图、箱线图 |
| 关联关系 | 热力图、散点图 |

## 配色方案

### 企业标准色
```css
--primary: #4264F5      /* 主色 */
--secondary: #6C7A89   /* 辅色 */
--success: #52C41A      /* 成功 */
--warning: #FAAD14      /* 警告 */
--error: #F5222D       /* 错误 */
```

### 图表配色顺序
1. #4264F5 (蓝)
2. #52C41A (绿)
3. #FAAD14 (黄)
4. #F5222D (红)
5. #722ED1 (紫)
6. #13C2C2 (青)

## 图表元素规范

### 标题
- 位置: 图表上方居中
- 字体: 16px 加粗
- 内容: 简洁描述图表主题

### 轴标签
- 字体: 12px
- 角度: 垂直时45度倾斜
- 单位: 必须标注 (如 "销售额(万元)")

### 图例
- 位置: 图表右侧或底部
- 交互: 支持点击筛选

### 数据标签
- 重要数据点必须显示数值
- 占比图显示百分比

## 导出规范

- 格式: PNG (优先)、SVG
- 分辨率: 2x (Retina)
- 尺寸: 16:9 (1200x675) 或 4:3 (800x600)
