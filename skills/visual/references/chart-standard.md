# 图表规范技能

## 核心功能

生成标准化图表，统一配色和格式规范

## 图表选择规范

| 数据特征 | 推荐图表 | 说明 |
|----------|----------|------|
| 趋势变化 | 折线图 | 展示随时间变化的趋势 |
| 占比分布 | 饼图/环形图 | 展示部分占总体的比例 |
| 对比分析 | 柱状图/条形图 | 展示不同类别的对比 |
| 关联关系 | 散点图 | 展示两个变量的关系 |
| 时间序列 | 面积图 | 展示累积变化 |
| 排名 | 排序柱状图 | 展示排名顺序 |
| 地理分布 | 地图 | 展示地域差异 |
| 热力分布 | 热力图 | 展示密度/强度 |

## 配色规范

### 企业主色
```css
--primary: #1E88E5;    /* 蓝色 - 主色 */
--secondary: #43A047;  /* 绿色 - 增长/正面 */
--warning: #FB8C00;    /* 橙色 - 警示 */
--error: #E53935;       /* 红色 - 下降/错误 */
--neutral: #757575;    /* 灰色 - 中性 */
```

### 图表配色序列
```
#1E88E5, #43A047, #FB8C00, #E53935, #8E24AA, #00ACC1
```

| 下降趋势 | #E53935 | #E57373 |
| 同比 | #1E88E5 | #90CAF9 |
| 环比 | #00ACC1 | #80DEEA |

## 图表规范

### 标题规范
- 位置：图表上方居中
- 字体：14px 粗体
- 内容：简洁明确

### 坐标轴规范
- 标签：12px 常规
- 刻度：10px
- 轴线：浅灰色 1px

### 图例规范
- 位置：图表右侧或底部
- 间距：8px
- 交互：支持点击筛选

### 数据标签
- 柱状图：顶部或柱内
- 折线图：点上方或尾部
- 饼图：扇区内或图例

## 输出结果

- 图表文件（PNG/SVG）
- 图表配置（JSON）
- 配色说明文档
- 图表清单

## [REVIEWER] 图表校验清单

### 生成图表后，必须校验以下项目：

| 校验项 | 标准 | 不通过处理 |
|--------|------|-----------|
| 标题 | 有标题且简洁明确 | 重新添加标题 |
| 坐标轴 | X/Y轴有标签和单位 | 补充轴标签 |
| 图例 | 有图例且位置正确 | 调整图例位置 |
| 数据标签 | 关键数据点有标签 | 添加数据标签 |
| 颜色 | 区分明显，不超过6色 | 调整配色方案 |
| 比例 | 无截断、无过度压缩 | 调整图表比例 |
| 文字 | 清晰可读，无遮挡 | 调整字体大小 |

### 校验流程

```markdown
生成图表 → [逐项校验] → 有不合格项? → 是 → 重新生成
                                              ↓ 否
                                        → 输出最终图表
```

### 不合格示例

- ❌ 标题缺失："2024销售"
- ❌ 坐标轴无单位："1000" 而非 "1000万元"
- ❌ 颜色过多：超过6个分类使用不同颜色
- ❌ 文字遮挡：图例被图表覆盖

### 合格示例

- ✅ 标题："2024年各月销售额趋势"
- ✅ 坐标轴："月份" / "销售额（万元）"
- ✅ 颜色：使用标准配色序列
- ✅ 文字：所有文字清晰可见
