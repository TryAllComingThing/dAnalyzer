# 数据分析技能 (Data Analysis)

## When to Activate

- Use this skill when analyzing data or performing data analysis
- Use this skill when extracting insights from data
- Use this skill when performing statistical analysis
- Use this skill when analyzing trends or patterns
- Use this skill when comparing data or distributions
- Use this skill when performing correlation analysis

## 分析类型

1. **描述性统计**：均值、中位数、最大最小值、标准差
2. **趋势分析**：时间序列趋势、环比/同比
3. **相关性分析**：变量间相关关系
4. **分布分析**：数据分布特征

## 执行流程

1. 数据理解与探索
2. 描述性统计分析
3. 趋势分析
4. 相关性分析
5. 分布分析
6. 洞察提炼

## 输出结果

- 分析报告
- 统计指标
- 趋势图表
- 洞察建议

## 依赖配置

- skills/data-query - 数据查询
- skills/data-clean - 数据清洗
- skills/visualization - 可视化
