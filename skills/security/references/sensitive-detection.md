# 敏感数据检测配置

## 核心功能

自动识别数据中的敏感字段和敏感内容

## 检测范围

### 1. 结构化数据（表格）

| 字段类型 | 检测规则 |
|----------|----------|
| 手机号 | 正则匹配 1[3-9]\d{9} |
| 身份证 | 正则匹配 \d{17}[\dXx] |
| 银行卡 | 正则匹配 \d{16,19} |
| 姓名 | 常见姓氏+名字模式 |
| 邮箱 | 正则匹配 .+@.+\..+ |
| 地址 | 包含省/市/区/路等 |

### 2. 非结构化数据（文本）

- 文档内容扫描
- 脱敏规则参考 skills/security/masking-rules.md

## 检测流程

1. 字段名分析
2. 数据内容采样检测
3. 正则匹配验证
4. 标记敏感字段

## 输出结果

- 敏感字段列表
- 敏感数据行号
- 脱敏建议
