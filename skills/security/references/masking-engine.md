# 脱敏引擎配置

## 核心功能

执行敏感数据的自动化脱敏处理

## 脱敏方法

### 1. 替换法

- 手机号：138****1234
- 身份证：110101****1234
- 银行卡：6222 **** **** 1234

### 2. 掩码法

- 姓名：张*
- 邮箱：z***@example.com
- 地址：北京市海淀区***

### 3. 截断法

- 金额：只保留万元
- 日期：只保留年月

### 4. 泛化法

- 年龄：25岁 → 20-30岁
- 收入：5000元 → 3000-5000元

## 执行流程

1. 读取原始数据
2. 识别敏感字段
3. 应用脱敏规则
4. 验证脱敏效果
5. 输出脱敏后数据

## 接口规范

```python
desensitize(data, rules):
    # data: 输入数据
    # rules: 脱敏规则配置
    return desensitized_data
```

## 配置项

- 脱敏方法选择
- 自定义规则
- 批量/逐条处理
- 日志记录

## 错误处理

- 脱敏失败：回滚原始数据
- 规则不匹配：跳过并记录
- 格式错误：报错提示
