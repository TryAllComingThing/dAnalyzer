# 脱敏规则库

## 核心功能

定义各类敏感数据的脱敏规则和格式

## 脱敏规则

### 1. 手机号码

- 格式：138****1234
- 规则：保留前3位和后4位，中间4位用*替换
- 正则：`^1[3-9]\d{9}$` → `$1****$2` (分组替换)

### 2. 身份证号码

- 格式：110101****1234
- 规则：保留前6位和后4位，中间8位用*替换
- 正则：`^\d{6}\d{8}\d{4}$` → `$1****$2`

### 3. 银行卡号

- 格式：6222 **** **** 1234
- 规则：保留前4位和后4位，中间用*替换
- 正则：`^\d{16,19}$` → `$1 **** **** $2`

### 4. 姓名

- 格式：张*、张**
- 规则：保留姓，名用*替换
- 2字姓名：张*
- 3字姓名：张**
- 4字姓名：张**

### 5. 邮箱

- 格式：z***@example.com
- 规则：@前保留首字符，中间用*替换

### 6. 地址

- 格式：北京市海淀区***
- 规则：保留省市，后面的详细信息用*替换

### 7. IP地址

- 格式：192.168.**.**
- 规则：保留前2段，后2段用*替换

### 8. 工资/营收

- 格式：¥****元
- 规则：具体金额用*替换，仅显示单位

## 脱敏流程

1. 识别敏感字段
2. 匹配脱敏规则
3. 执行脱敏转换
4. 记录脱敏日志

## 优先级

1. legal规则优先（身份证、手机号）
2. 核心规则次之（银行卡、姓名）
3. 基础规则补充（地址、邮箱）
