# 敏感字段脱敏技能

## 核心功能

识别敏感字段并按规则自动脱敏

## 识别字段类型

| 字段类型 | 识别规则 | 敏感等级 |
|----------|----------|----------|
| 手机号 | 11位数字，以13/14/15/16/17/18/19开头 | 高 |
| 身份证号 | 15/18位，身份证号格式 | 高 |
| 银行卡号 | 13-19位数字 | 高 |
| 姓名 | 常见姓氏+名字 | 中 |
| 邮箱 | xxx@xxx.xx格式 | 中 |
| 地址 | 包含省市区关键字 | 中 |
| IP地址 | xxx.xxx.xxx.xxx格式 | 低 |

## 脱敏规则

| 字段 | 脱敏方式 | 示例 |
|------|----------|------|
| 手机号 | 中间4位脱敏 | 138****1234 |
| 身份证 | 出生日期脱敏 | 110101****1234 |
| 银行卡 | 卡号段脱敏 | 6222 **** **** 1234 |
| 姓名 | 姓氏保留，名字脱敏 | 张**、李* |
| 邮箱 | 用户名脱敏 | us***@***.com |
| 地址 | 省市保留，详细地址脱敏 | 北京市*** |
| IP地址 | 最后一段脱敏 | 192.168.0.*** |

## 执行流程

1. 扫描数据字段
2. 识别敏感字段类型
3. 应用脱敏规则
4. 验证脱敏效果
5. 生成脱敏报告

## 输出结果

- 脱敏后数据文件
- 脱敏日志
- 脱敏报告（脱敏字段清单、脱敏前后对比）
