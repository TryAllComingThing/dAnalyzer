# 预测模型技能

## 核心功能

基于历史数据进行预测

## 预测方法

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 移动平均 | 短期预测 | 简单 | 滞后 |
| 指数平滑 | 趋势预测 | 灵活 | 参数敏感 |
| 线性回归 | 线性关系 | 可解释 | 线性假设 |
| ARIMA | 时间序列 | 精确 | 参数复杂 |

## 评估指标

| 指标 | 说明 |
|------|------|
| MAE | 平均绝对误差 |
| RMSE | 均方根误差 |
| MAPE | 平均百分比误差 |
