# 归因分析技能

## 核心功能

分析各因素对结果的贡献

## 归因模型

| 模型 | 说明 | 适用场景 |
|------|------|----------|
| 首触归因 | 100%归因首个触点 | 新客获取 |
| 末触归因 | 100%归因最后触点 | 转化优化 |
| 线性归因 | 均分各触点 | 品牌建设 |
| 时间衰减 | 越近触点权重越高 | 短期转化 |
| 位置归因 | 首末各40%中间20% | 综合评估 |

## 计算方法

```python
# 线性归因
contribution = 1 / touchpoints_count

# 时间衰减归因
contribution = decay_factor ^ (position_from_end)
```

## 输出结果

- 归因分析报告
- 各触点贡献度
- 优化建议
