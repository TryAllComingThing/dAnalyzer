# RFM参数咨询阶段

## 目的
在执行RFM分析前，必须明确分析的时间范围和RFM阈值参数。

## 必须确认的问题

### 1. 分析时间范围
- 统计周期开始日期: _______
- 统计周期结束日期: _______

### 2. R 参数（最近消费时间）
```
当前设定（可调整）:
  5分: ≤30天
  4分: 31-90天
  3分: 91-180天
  2分: 181-365天
  1分: >365天

您的调整: _______
```

### 3. F 参数（消费频率）
```
当前设定（可调整）:
  5分: ≥10次
  4分: 6-9次
  3分: 3-5次
  2分: 2次
  1分: 1次

您的调整: _______
```

### 4. M 参数（消费金额）
```
当前设定（可调整）:
  5分: ≥5000元
  4分: 2000-4999元
  3分: 500-1999元
  2分: 100-499元
  1分: <100元

您的调整: _______
```

### 5. 业务标识
- 消费类型: 全部/仅付费/仅退款
- 用户状态: 全部/仅活跃/仅流失

### 6. 分析目标
```
□ 用户分层统计
□ 高价值用户画像
□ 流失用户预警
□ 精准营销建议
全部/选择: _______
```

## 咨询模板

```
开始RFM分析前，请确认以下信息：

1. 分析的时间范围？（如：2025年1月-12月）
2. R/F/M 的阈值是否使用默认值？如有调整请说明
3. 主要分析目标是什么？（用户分层/高价值用户/流失预警）
4. 是否有特定的用户群体需要过滤？
```

## 规则

- **未确认以上信息前，禁止开始数据分析**
- 如果用户选择默认参数，也需要明确确认
- 确认后记录到执行上下文
