# 趋势分析技能

## 核心功能

分析数据随时间变化的趋势

## 分析类型

### 趋势类型
- 上升趋势
- 下降趋势
- 平稳趋势
- 波动趋势
- 季节性波动

### 对比类型
- 环比：与上期对比
- 同比：与去年同期对比
- 定比：与固定基准对比

## 分析指标

| 指标 | 说明 |
|------|------|
| 增长量 | 本期值 - 上期值 |
| 增长率 | (本期值 - 上期值) / 上期值 |
| 累计增长 | 期末值 - 期初值 |
| 平均增长率 | 几何平均增长率 |

## 输出结果

- 趋势数据表
- 趋势图表
- 趋势分析报告
- 预测建议
