# 模型选择规则 (Model Selection Reviewer)

## 目的

作为Reviewer，根据数据特征和业务场景自动推荐最佳分析模型

## 选择规则

### 数据特征分析

| 特征 | 推荐模型 |
|------|----------|
| 数据量 > 10万条 | 聚类分析/预测模型 |
| 数据量 < 1000条 | 统计检验为主 |
| 包含时间序列 | 趋势分析/预测模型 |
| 包含多个渠道字段 | 归因分析 |
| 包含R/F/M字段 | RFM模型 |
| 包含转化步骤 | 漏斗模型/队列分析 |

### 业务场景匹配

| 业务场景 | 推荐模型 |
|----------|----------|
| 用户价值分层 | RFM模型 |
| 用户行为分群 | 聚类分析 |
| 渠道效果评估 | 归因分析 |
| 转化路径优化 | 漏斗模型 |
| 销售趋势预测 | 预测模型 |
| 留存分析 | 队列分析 |
| 变量关系分析 | 相关性分析 |

### 字段类型检测

| 字段类型 | 适用模型 |
|----------|----------|
| 连续数值型 | 相关性分析、聚类、预测 |
| 分类字段 | 聚类、归因 |
| 日期时间型 | 趋势分析、队列分析 |
| 布尔型 | 归因、漏斗 |

## 校验规则

### 输入校验

- 数据量必须 > 0
- 关键字段不能全部为空
- 数值字段需检查是否为有效数字

### 输出校验

| 模型类型 | 校验规则 |
|----------|----------|
| 聚类 | 簇内误差平方和需下降 |
| 归因 | 各渠道贡献度和 = 100% |
| 预测 | 预测值在合理范围内 |
| 漏斗 | 转化率递减 |

## 推荐输出格式

```markdown
## 模型推荐结果

**推荐模型**: [模型名称]
**推荐理由**:
1. 数据特征匹配度: XX%
2. 业务场景匹配度: XX%
3. 历史使用效果: XX%

**备选模型**:
- [备选1] - 适用场景...
- [备选2] - 适用场景...

**请确认**: 是否使用推荐的 [模型名称]？或选择其他模型？
```
