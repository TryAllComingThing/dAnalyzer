# 相关性分析技能

## 核心功能

分析变量之间的相关关系

## 相关类型

### 线性相关
- 正相关：一个变量增加，另一个也增加
- 负相关：一个变量增加，另一个减少
- 无相关：变量之间无明显关系

### 相关系数
| 系数范围 | 相关程度 |
|----------|----------|
| 0.8-1.0 | 强正相关 |
| 0.5-0.8 | 中等正相关 |
| 0.3-0.5 | 弱正相关 |
| -0.3-0.3 | 几乎无相关 |
| -0.5--0.3 | 弱负相关 |
| -0.8--0.5 | 中等负相关 |
| -1.0--0.8 | 强负相关 |

## 分析方法

- Pearson相关系数
- Spearman秩相关
- Kendall相关

## 输出结果

- 相关系数矩阵
- 相关性图表（热力图）
- 显著相关变量对清单
- 分析结论
