# 聚合统计查询技能

## 核心功能

执行聚合统计分析

## 聚合类型

| 类型 | 函数 | 说明 |
|------|------|------|
| 求和 | SUM | 数值求和 |
| 计数 | COUNT | 记录数 |
| 均值 | AVG | 平均值 |
| 最大 | MAX | 最大值 |
| 最小 | MIN | 最小值 |
| 去重计数 | COUNT DISTINCT | 去重后计数 |
| 合计 | COUNT(*) | 总数 |

## 分组维度

- 时间维度：按日/周/月/季度/年
- 渠道维度：按APP/Web/小程序等
- 品类维度：按一/二/三级品类
- 地域维度：按省/市/区域
- 用户维度：按用户类型/等级

## 输入参数

- 数据表
- 分组维度（多个）
- 聚合指标（多个）
- 过滤条件
- 排序要求

## 输出结果

- 聚合结果表
- 统计日志
