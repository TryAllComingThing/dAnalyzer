# 周报生成技能

## 核心功能

按业务线、时间范围自动生成标准化周报

## 输入参数

- 业务线
- 时间范围（周/月）
- 数据文件路径

## 报告结构

1. 标题（业务线+周期）
2. 核心指标（销售额、订单量、客单价等）
3. 核心发现（关键变化）
4. 异常分析（异常指标说明）
5. 下周建议（运营建议）

## 调用资产

- assets/template/report-template.md
- assets/indicator/core-indicator-dict.md

## 输出结果

- 周报文件（Markdown格式）
- 图表文件（可选）
- 生成日志
