# 重复值检测与去重

## 功能
检测并处理数据中的重复记录

## 重复类型

| 类型 | 描述 | 处理策略 |
|------|------|----------|
| 完全重复 | 所有字段相同 | 保留一条 |
| 主键重复 | 关键字段相同 | 保留最新/最早 |
| 部分重复 | 部分字段相同 | 合并/聚合 |

## 检测规则
1. 指定主键字段
2. 识别重复组
3. 选择保留策略

## 保留策略
- **保留最新**: 按时间戳保留最新记录
- **保留最早**: 保留首次出现的记录
- **保留最完整**: 保留空值最少的记录
- **合并**: 聚合重复记录的数值字段

## 执行步骤
1. 定义主键
2. 识别重复
3. 选择策略
4. 执行去重
5. 输出报告
