# 异常值处理

## 功能
识别和处理数据中的异常值

## 检测方法

### 1. 3σ原则
- 适用于正态分布数据
- 超出 [μ-3σ, μ+3σ] 范围的值视为异常

### 2. IQR四分位法
- 计算 Q1 (25%) 和 Q3 (75%)
- IQR = Q3 - Q1
- 异常值: < Q1 - 1.5*IQR 或 > Q3 + 1.5*IQR

### 3. 业务规则
- 根据业务知识定义合理范围
- 例如: 年龄应在 0-150 岁

## 处理方式
1. **标记**: 添加标识字段记录异常
2. **剔除**: 删除异常记录
3. **替换**: 用边界值或均值替换
4. **修正**: 根据业务规则修正

## 执行步骤
1. 选择检测方法
2. 识别异常值
3. 选择处理策略
4. 执行处理
5. 记录日志
