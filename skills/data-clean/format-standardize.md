# 数据格式标准化

## 功能
统一数据格式，确保一致性

## 标准化规则

### 1. 日期格式
| 类型 | 标准格式 |
|------|----------|
| 日期 | YYYY-MM-DD |
| 时间 | HH:MM:SS |
| 日期时间 | YYYY-MM-DD HH:MM:SS |

### 2. 数值格式
- 小数位数: 统一保留2位
- 千分位: 使用逗号分隔
- 货币: ¥符号 + 金额

### 3. 字符编码
- 统一: UTF-8
- 特殊字符转义

### 4. 枚举值
- 统一大小写
- 去除前后空格
- 标准化缩写

## 执行步骤
1. 识别各字段类型
2. 应用标准化规则
3. 验证格式
4. 输出标准化数据
