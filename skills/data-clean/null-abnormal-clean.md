# 空值与异常值清洗

## 功能
处理数据中的空值和缺失数据

## 处理策略

### 数值型字段
| 情况 | 处理方式 |
|------|----------|
| 少量空值 (<5%) | 均值填充 |
| 中等空值 (5%-20%) | 中位数填充 |
| 大量空值 (>20%) | 标记为缺失类别 |

### 字符型字段
| 情况 | 处理方式 |
|------|----------|
| 少量空值 | "未知"填充 |
| 分类字段 | 众数填充 |
| 关键字段 | 单独标记 |

## 执行步骤
1. 统计各字段空值比例
2. 选择填充策略
3. 执行填充
4. 记录处理日志
