# /analysis rfm - RFM 用户分析

## 功能说明

RFM 用户价值分析模型:
- **R**ecency: 最近消费时间 (流失可能性)
- **F**requency: 消费频率 (忠诚度)
- **M**onetary: 消费金额 (价值贡献)

## 使用方式

```
/analysis rfm <用户表> [--time_window=<月数>] [--segments=<分群数>]
```

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| 用户表 | 是 | 用户行为数据表名 |
| --time_window | 否 | 分析时间窗口 (默认6个月) |
| --segments | 否 | 用户分群数 (默认5) |

## 使用示例

### 基础分析

```bash
# 6个月RFM分析
/analysis rfm user_orders

# 3个月RFM分析
/analysis rfm user_orders --time_window=3

# 自定义分群
/analysis rfm user_orders --segments=3
```

### 高级分析

```bash
# 指定日期
/analysis rfm user_orders --reference=2026-04-01

# 包含字段
/analysis rfm user_orders --recency=last_order_time --frequency=order_count --monetary=total_amount
```

## 用户分群说明

| 用户群 | R | F | M | 营销策略 |
|--------|---|---|---|----------|
| 核心客户 | 高 | 高 | 高 | VIP服务、优先体验 |

| 价值客户 | 高 | 低 | 高 | 提升消费频次 |
| 新客户 | 高 | 低 | 低 | 新人礼包、引导复购 |
| 重要挽留 | 低 | 高 | 高 | 召回活动、专属优惠 |
| 流失预警 | 低 | 高 | 低 | 流失预警、挽留 |

| 流失客户 | 低 | 低 | 低 | 召回成本高 |

## 自动处理

1. **RFM 计算**: 自动计算三个维度分数
2. **用户分群**: 基于分数自动分群
3. **风险识别**: 识别高流失风险用户
4. **营销建议**: 针对不同群体给出建议
