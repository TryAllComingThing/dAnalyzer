# /analysis audit - 审计分析

## 功能说明

分析系统操作审计日志:
- 用户操作记录
- 数据变更追踪
- 异常行为检测
- 合规审计

## 使用方式

```
/analysis audit [--user=<用户>] [--action=<操作>] [--time=<时间范围>]
```

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| --user | 否 | 用户ID |
| --action | 否 | 操作类型 (create/update/delete/export) |
| --time | 否 | 时间范围 (today/yesterday/7d/30d) |

## 使用示例

### 基本审计

```bash
# 今日审计
/analysis audit --time=today

# 昨天审计
/analysis audit --time=yesterday

# 7天审计
/analysis audit --time=7d
```

### 用户审计

```bash
# 指定用户审计
/analysis audit --user=U12345

# 用户导出操作审计
/analysis audit --user=U12345 --action=export
```

### 操作审计

```bash
# 数据导出审计
/analysis audit --action=export

# 数据删除审计
/analysis audit --action=delete

# 敏感操作审计
/analysis audit --action=sensitive
```

## 操作类型

| 类型 | 说明 | 风险等级 |
|------|------|----------|
| create | 新增操作 | 低 |
| update | 更新操作 | 中 |
| delete | 删除操作 | 高 |
| export | 导出操作 | 中 |
| query | 查询操作 | 低 |
| login | 登录操作 | 中 |
| sensitive | 敏感操作 | 高 |

## 自动处理

1. **日志收集**: 自动收集各系统审计日志
2. **异常检测**: 识别异常操作模式
3. **风险评分**: 评估操作风险等级
4. **告警**: 高风险操作实时告警

## 合规要求

- ✅ 符合审计日志保留要求 (至少6个月)
- ✅ 支持数据溯源
- ✅ 支持合规报告生成
