# /analysis funnel - 漏斗分析

## 功能说明

分析用户转化路径，识别流失环节:
- 各阶段转化率
- 总体转化率
- 流失原因分析
- 优化建议

## 使用方式

```
/analysis funnel <漏斗定义> [--time_window=<时间范围>]
```

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| 漏斗定义 | 是 | 漏斗步骤，用→分隔 |
| --time_window | 否 | 分析时间范围 |

## 使用示例

### 标准漏斗

```bash
# 电商购买漏斗
/analysis funnel 访问→浏览→加购→下单→支付→成交

# 注册转化漏斗
/analysis funnel 注册→激活→首次下单

# 简单漏斗
/analysis funnel 访问→注册
```

### 自定义漏斗

```bash
# 指定时间
/analysis funnel 访问→成交 --time_window=30d

# 电商完整漏斗
/analysis funnel 曝光→点击→浏览→加购→下单→支付完成
```

## 漏斗阶段说明

```
标准电商漏斗:
访问 → 浏览商品 → 加购物车 → 提交订单 → 支付 → 成交

标准注册漏斗:
访问 → 注册 → 完善资料 → 首次访问 → 首次成交
```

## 自动处理

1. **数据计算**: 自动计算各阶段转化率
2. **可视化**: 自动生成漏斗图
3. **分析**: 识别最大流失环节
4. **建议**: 基于行业最佳实践给出优化建议

## 注意事项

1. **步骤顺序**: 确保漏斗步骤逻辑正确
2. **时间窗口**: 大数据量建议指定时间范围
3. **数据源**: 确保各阶段数据可关联
