# /report weekly - 周报生成

## 功能说明

生成每周运营数据报告，包括:
- 周度数据汇总
- 趋势分析
- 竞品对比 (如有)
- 下周预测

## 使用方式

```
/report weekly <业务线> [--template=<模板>] [--include_forecast]
```

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| 业务线 | 是 | sales/user/product/activity |
| --template | 否 | 报告模板 (default/detailed/simple) |
| --include_forecast | 否 | 包含下周预测 |

## 使用示例

### 基本用法

```bash
# 销售周报
/report weekly sales

# 用户周报
/report weekly user

# 产品周报
/report weekly product
```

### 高级用法

```bash
# 含预测
/report weekly sales --include_forecast

# 详细周报
/report weekly sales --template=detailed
```

## 自动包含内容

1. **周度汇总**: 7天累计数据
2. **趋势图表**: 每日趋势可视化
3. **排名分析**: 品类/地区/渠道排名
4. **用户分析**: 新增/活跃/复购
5. **周度洞察**: 数据解读和建议
6. **预测** (可选): 下周预测

## 与日报对比

| 维度 | 日报 | 周报 |
|------|------|------|
| 时间范围 | 当日 | 7天 |
| 分析深度 | 基础 | 深入 |
| 趋势图表 | 较少 | 完整 |
| 预测 | 无 | 有 |
| 洞察 | 简单 | 详细 |

## 相关命令

- `/report daily` - 日报
- `/report custom` - 自定义报告
- `/analysis trend` - 趋势分析
- `/help report` - 返回报告命令列表
