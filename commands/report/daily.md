# /report daily - 日报生成

## 功能说明

生成每日运营数据报告，包括:
- 核心指标概览
- 关键数据对比
- 异常告警
- 趋势洞察

## 使用方式

```
/report daily <业务线> [--template=<模板>]
```

## 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| 业务线 | 是 | sales/user/product/activity |
| --template | 否 | 报告模板 (default/detailed/simple) |

## 使用示例

### 基本用法

```bash
# 销售日报
/report daily sales

# 用户日报
/report daily user

# 产品日报
/report daily product

# 活动日报
/report daily activity
```

### 自定义模板

```bash
# 详细日报
/report daily sales --template=detailed

# 简洁日报
/report daily sales --template=simple
```

## 自动包含内容

1. **核心指标**: 销售额、订单、客单价、转化率
2. **维度分析**: 品类、地区、渠道
3. **趋势对比**: 环比、同比
4. **异常检测**: 指标异常告警
5. **洞察建议**: 数据解读和建议

## 定时任务 (可选)

```bash
# 设置自动生成日报
/report daily sales --schedule=09:00

# 取消自动日报
/report daily sales --schedule=none
```
