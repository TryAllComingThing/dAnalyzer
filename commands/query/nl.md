# /query nl - 自然语言查询

## 功能说明

使用自然语言描述查询需求，系统自动转换为 SQL 并执行。适合不熟悉 SQL 的用户。

## 使用方式

```
/query nl <自然语言描述>
```

## 使用示例

### 数据查询

```bash
# 查询销售额
/query nl 本月销售额

# 查询订单数
/query nl 过去7天每天的新增用户数

# 带条件的查询
/query nl 华北区销售额排名前10的客户

# 时间范围查询
/query nl 2026年Q1每月销量
```

### 数据统计

```bash
# 统计类
/query nl 平均客单价是多少

# 占比类
/query nl 各产品类别销量占比

# 对比类
/query nl 同比增长率最高的产品
```

### 复杂查询

```bash
# 多维度
/query nl 按地区和月份的销售额分布

# 筛选+排序
/query nl 转化率最低的10个商品

# 关联查询
/query nl 购买金额最高的前100名用户的详细信息
```

## 输出示例

```
用户输入: 本月销售额

→ 自动生成 SQL:
SELECT SUM(amount) as revenue FROM orders
WHERE order_date >= '2026-04-01' AND order_date < '2026-05-01'

执行结果:
┌──────────┐
│  销售额  │
├──────────┤
│ 1,234,567元 │
└──────────┘
```

## 自动处理

1. **意图识别**: 判断查询意图 (统计/排序/筛选)
2. **SQL 生成**: 自动转换为 SQL
3. **行业适配**: 根据行业上下文优化查询
4. **结果呈现**: 智能选择展示格式 (表格/图表)
5. **安全检查**: 自动脱敏 + 合规检查

## 支持的模式

| 模式 | 示例 | 说明 |
|------|------|------|
| 时间查询 | 本月/上周/去年Q1 | 自动识别时间范围 |
| 排序查询 | 前10/倒数 | 自动添加 LIMIT |
| 筛选查询 | 销售额>1000 | 自动添加 WHERE |
| 统计查询 | 总和/平均/占比 | 自动选择聚合函数 |
| 对比查询 | 同比/环比 | 自动生成对比 SQL |

## 注意事项

1. **描述尽量具体**: 越详细的描述，SQL 越准确
2. **明确时间范围**: 建议包含明确的时间表达
3. **结果确认**: 复杂查询建议确认 SQL 后再执行

## 相关命令

- `/query sql` - SQL 直接查询
- `/query export` - 数据导出
- `/chart` - 生成图表
- `/help query` - 返回查询命令列表
