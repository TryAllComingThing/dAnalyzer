# 文件命名规范

## 核心职责

规范输出文件的命名

## 命名规则

### 数据文件

```
[业务线]_[数据类型]_[时间范围]_[版本].[格式]

示例：
电商_订单数据_20260401-20260407_v1.csv
电商_用户数据_2026年4月_v1.xlsx
```

### 报告文件

```
[业务线]_[报告类型]_[周期]_[版本].[格式]

示例：
电商_周报_2026年第17周_v1.md
电商_月报_2026年4月_v1.pdf
```

### 图表文件

```
[业务线]_[图表类型]_[时间范围]_[序号].[格式]

示例：
电商_销售趋势图_20260401-20260407_01.png
电商_用户分布图_2026年4月_01.svg
```

## 禁止命名

- 禁止使用中文特殊字符
- 禁止空格（用下划线替代）
- 禁止过长命名（不超过50字符）
- 禁止临时/测试等字样

## 校验方式

CLI提醒式校验，仅给出优化建议
