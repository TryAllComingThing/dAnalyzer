# 代码注释规范

## 核心职责

规范代码和SQL的注释要求

## 注释要求

### SQL注释

```sql
-- 功能：查询订单销售统计
-- 作者：数据团队
-- 日期：2026-04-24
-- 说明：统计周期T+1数据
SELECT
    dt,                       -- 日期
    COUNT(*) AS order_cnt,    -- 订单数
    SUM(amount) AS amount     -- 销售金额
FROM order_info
WHERE dt = '${date}'
```

### 文件头注释

```markdown
---
文件：order-stat.md
功能：订单统计查询
作者：数据团队
更新：2026-04-24
---
```

### 字段注释

```sql
-- 金额字段单位：元
-- 时间字段单位：秒
-- 数量字段单位：个
```

## 校验方式

CLI提醒式校验，仅给出优化建议
