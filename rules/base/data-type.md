# 数据类型规范

## 核心职责

统一数据字段的类型定义

## 数据类型标准

### 数值类型

| 类型 | 说明 | 范围 |
|------|------|------|
| INT | 整数 | -21亿~21亿 |
| BIGINT | 大整数 | -900京~900京 |
| DECIMAL(18,2) | 精确数值 | 金额 |
| FLOAT | 浮点数 | 近似值 |

### 字符串类型

| 类型 | 说明 | 长度 |
|------|------|------|
| VARCHAR(50) | 变长字符串 | <=50 |
| VARCHAR(255) | 变长字符串 | <=255 |
| TEXT | 长文本 | 无限制 |
| ENUM | 枚举值 | 固定值 |

### 日期类型

| 类型 | 说明 | 格式 |
|------|------|------|
| DATE | 日期 | YYYY-MM-DD |
| DATETIME | 日期时间 | YYYY-MM-DD HH:MM:SS |
| TIMESTAMP | 时间戳 | Unix时间戳 |

### 布尔类型

| 类型 | 说明 |
|------|------|
| BOOLEAN | true/false |
| TINYINT(1) | 0/1 |

## 业务类型映射

| 业务字段 | 推荐类型 | 说明 |
|---------|----------|------|
| 金额 | DECIMAL(18,2) | 精确到分 |
| 数量 | INT/BIGINT | 整数 |
| 比率 | DECIMAL(5,4) | 0.0000-1.0000 |
| 手机号 | VARCHAR(11) | 11位 |
| 身份证 | VARCHAR(18) | 15/18位 |
| 状态 | TINYINT | 0/1/2 |

## 校验方式

CLI提醒式校验

## 处理方式

- 类型不匹配 → 预警
- 建议类型 → 提示
