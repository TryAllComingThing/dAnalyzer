# 报告格式规范

## 核心职责

统一分析报告的格式标准

## 报告结构

### 标准报告结构

```
# 标题
## 一、概览/摘要
## 二、核心指标
## 三、详细分析
### 3.1 [维度1]分析
### 3.2 [维度2]分析
## 四、问题与机会
## 五、建议/结论
---
元信息：报告时间、数据周期、编制人
```

## 格式要求

### 标题规范
- 一级标题：# 标题（报告名称）
- 二级标题：## 大章节
- 三级标题：### 小章节

### 表格规范
- 使用Markdown表格
- 表头加粗
- 数值右对齐
- 文字左对齐

### 图表规范
- 图表需有标题
- 图表需有坐标轴标签
- 图表需有图例

### 数字格式
- 金额：千分位分隔，保留2位
- 百分比：保留2位小数
- 数量：千分位分隔

### 时间格式
- 日期：YYYY-MM-DD
- 周：YYYY年第N周
- 月：YYYY年M月

## 校验方式

CLI提醒式校验，仅给出优化建议
