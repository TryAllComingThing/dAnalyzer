# 出口数据管控规则

## 核心职责

管控数据出口，防止敏感数据外泄

## 管控范围

### 敏感字段管控

| 字段类型 | 管控级别 | 出口要求 |
|----------|----------|----------|
| 身份证号 | 严格管控 | 禁止出口 |
| 手机号 | 严格管控 | 脱敏出口 |
| 银行卡号 | 严格管控 | 禁止出口 |
| 姓名 | 管控 | 脱敏出口 |
| 详细地址 | 管控 | 脱敏出口 |

### 数据量管控

- 单次导出≤10000条
- 日累计导出≤100000条
- 超过阈值需审批

### 字段管控

- 禁止导出全部字段
- 必脱敏字段：敏感字段
- 建议脱敏字段：准敏感字段

## 审批流程

### 批量数据导出
1. 部门负责人审批
2. 数据安全审批
3. 记录导出日志

### 敏感数据导出
1. 数据安全团队审批
2. 主管领导审批
3. 记录导出日志
4. 加密传输

## 校验方式

- 前置校验：导出请求校验
- 后置校验：导出数据扫描

## 处理方式

- 未脱敏 → 拦截+预警
- 超量导出 → 预警+审批
- 敏感字段 → 禁止导出
