# 同期群模型资产

## 模型定义

按用户首次行为时间分组，分析用户群的后续行为特征

## 模型结构

### 按月份分组
- 2026年1月新增用户群
- 2026年2月新增用户群
- 2026年3月新增用户群
- ...

### 分析维度
- 留存率（次月、3月、6月...）
- 收入贡献
- 行为特征

## 计算公式

```
同期群留存率 = 某月新增用户中在后续N月有行为的用户数 / 该月新增用户总数
```

## 适用场景

- 用户生命周期分析
- 获客质量评估
- 产品迭代效果评估
- 运营策略效果评估

## 输出指标

| 指标 | 说明 |
|------|------|
| 群组 | 用户首次行为月份 |
| 规模 | 新增用户数 |
| 次月留存 | 1个月后留存率 |
| 3月留存 | 3个月后留存率 |
| 6月留存 | 6个月后留存率 |
| 累计收入 | 群组累计贡献收入 |

## 分析建议

- 对比不同群组留存曲线
- 识别留存拐点
- 分析影响留存的因
