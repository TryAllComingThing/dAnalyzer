# 用户分群模型资产

## 模型定义

基于多维度对用户进行分群

## 分群方式

### 规则分群
基于业务规则进行分群
- 价值分群：高/中/低价值
- 活跃分群：高/中/低活跃
- 生命周期：新/老/流失

### 行为分群
基于用户行为特征
- 浏览型用户
- 加购型用户
- 购买型用户
- 复购型用户

### 兴趣分群
基于用户兴趣偏好
- 品类偏好
- 价格敏感度
- 品牌偏好

### RFM分群
基于RFM模型
- 高价值用户
- 潜力用户
- 流失风险
- 休眠用户

## 分群维度

| 维度 | 说明 | 示例值 |
|------|------|--------|
| 消费金额 | 累计消费 | 1000元 |
| 消费频次 | 订单数量 | 5次 |
| 最近消费 | 距今天数 | 7天 |
| 品类偏好 | 偏好品类 | 服装 |
| 价格敏感 | 价格敏感度 | 高/中/低 |

## 分群应用

| 群体 | 运营策略 |
|------|----------|
| 高价值+高活跃 | VIP服务，专属活动 |
| 高价值+低活跃 | 唤醒策略 |
| 低价值+高活跃 | 提升消费 |
| 低价值+低活跃 | 低成本维护 |

## 输出指标

- 用户分群结果表
- 各群用户统计
- 各群特征描述
- 运营建议
