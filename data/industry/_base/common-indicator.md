# 行业基础配置

> 所有行业共用的通用配置

## 通用指标

所有行业都可能使用的通用维度：

### 时间维度

| 维度 | 说明 | 示例值 |
|------|------|--------|
| 日期 | 具体日期 | 2026-04-25 |
| 周 | 一年中的第几周 | 2026-W17 |
| 月 | 月份 | 2026-04 |
| 季度 | 季度 | 2026-Q2 |
| 年 | 年份 | 2026 |

### 组织维度

| 维度 | 说明 |
|------|------|
| 公司 | 公司/集团 |
| 部门 | 组织部门 |
| 团队 | 小组/团队 |
| 员工 | 人员 |

### 通用指标

| 指标 | 说明 |
|------|------|
| 数量 | 记录总数 |
| 金额 | 金额相关 |
| 占比 | 百分比 |
| 变化 | 环比/同比增长 |

## 通用分析维度

- 时间维度：日、周、月、季度、年
- 地域维度：国家、省份、城市、区域
- 渠道维度：线上、线下、直营、代理
- 客户维度：个人、企业
