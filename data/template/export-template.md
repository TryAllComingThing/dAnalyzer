# 导出格式模板

## CSV导出模板

```csv
字段1,字段2,字段3
值1,值2,值3
```

## Excel导出模板

```markdown
### 表头
| 字段1 | 字段2 | 字段3 |
|-------|-------|-------|
| 值1   | 值2   | 值3   |

### 多Sheet
- Sheet1: 汇总数据
- Sheet2: 明细数据
```

## JSON导出模板

```json
{
  "meta": {
    "export_time": "2026-04-24",
    "record_count": 1000
  },
  "data": [
    {"field1": "value1", "field2": "value2"}
  ]
}
```

## 导出配置

| 配置项 | 说明 |
|--------|------|
| 编码 | UTF-8 |
| 分隔符 | 逗号 |
| 换行 | LF |
| 引号 | 双引号 |
| 压缩 | GZIP可选 |
