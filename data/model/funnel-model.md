# 漏斗模型资产

## 模型定义

分析用户从入口到转化的各阶段转化率，识别流失环节

## 漏斗阶段

### 电商漏斗
1. **浏览（View）**：用户访问商品页/首页
2. **加购（Cart）**：用户将商品加入购物车
3. **下单（Order）**：用户提交订单
4. **付款（Payment）**：用户完成支付
5. **复购（Repurchase）**：用户再次购买

### 注册漏斗
1. **注册**：用户点击注册
2. **实名**：完成实名认证
3. **绑卡**：绑定银行卡
4. **首次投资**：完成首次投资

### 营销漏斗
1. **曝光**：活动曝光
2. **点击**：点击参与
3. **领取**：领取优惠券
4. **使用**：使用优惠券

## 计算公式

```
阶段转化率 = 当前阶段用户数 / 上一阶段用户数
总体转化率 = 最终阶段用户数 / 第一阶段用户数
流失率 = 1 - 阶段转化率
流失用户数 = 上一阶段用户数 - 当前阶段用户数
```

## 示例数据

| 阶段 | 用户数 | 阶段转化率 | 流失率 |
|------|--------|------------|--------|
| 浏览 | 10000 | - | - |
| 加购 | 3000 | 30% | 70% |
| 下单 | 1500 | 50% | 50% |
| 付款 | 1000 | 66.7% | 33.3% |

## 适用场景

- 电商转化分析
- 注册流程优化
- 营销活动效果评估
- 产品功能迭代
- 用户体验优化

## 输出指标

| 指标 | 说明 |
|------|------|
| 各阶段用户数 | 每个阶段的独立用户数 |
| 各阶段转化率 | 当前/上一阶段用户数 |
| 总体转化率 | 最终/第一阶段用户数 |
| 流失用户数 | 各阶段流失的用户数 |
| 流失率 | 流失用户数/上一阶段 |
| 平均转化时间 | 各阶段平均耗时 |

## 优化建议

- 识别转化率最低的阶段
- 分析流失原因
- 制定优化策略
- 验证优化效果
