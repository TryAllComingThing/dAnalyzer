# AARRR模型资产

## 模型定义

AARRR是用户增长的核心模型，包含5个关键指标

## 模型结构

### Acquisition（获取）
- 用户如何发现产品？
- 渠道来源
- 获客成本

### Activation（激活）
- 用户是否体验到产品核心价值？
- 首次使用转化
- 关键行为完成率

### Retention（留存）
- 用户是否持续使用？
- 留存率曲线
- 用户生命周期

### Referral（推荐）
- 用户是否愿意推荐？
- 分享率
- NPS评分

### Revenue（收入）
- 如何从用户身上盈利？
- 付费率
- ARPU
- LTV

## 指标体系

| 阶段 | 核心指标 | 计算方式 |
|------|----------|----------|
| 获取 | 渠道曝光量 | SUM(exposure) |
| 获取 | 渠道点击量 | SUM(click) |
| 获取 | 获客成本 | 渠道花费/新用户数 |
| 激活 | 首次转化率 | 首次关键行为用户/新用户 |
| 留存 | 次日留存 | 次日回访/新增 |
| 留存 | 7日留存 | 7日回访/新增 |
| 留存 | 30日留存 | 30日回访/新增 |
| 推荐 | 分享率 | 分享用户/活跃用户 |
| 推荐 | NPS | 推荐者-贬损者 |
| 收入 | 付费率 | 付费用户/活跃用户 |
| 收入 | ARPU | 收入/付费用户 |

## 适用场景

- 用户增长分析
- 渠道效果评估
- 产品体验优化
- 商业化分析

## 优化方向

- 优化获客渠道
- 提升激活转化
- 提高用户留存
- 增强推荐意愿
- 优化收入模型
