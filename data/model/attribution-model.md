# 归因模型资产

## 模型定义

分析各触点对转化的贡献

## 模型类型

### 首触归因
- 100%归因首个触点
- 适用：品牌建设、认知阶段

### 末触归因
- 100%归因最后触点
- 适用：转化优化、短期效果

### 线性归因
- 均分各触点
- 适用：品牌建设

### 时间衰减归因
- 越近触点权重越高
- 适用：短期转化

### 位置归因
- 首末各40%，中间均分20%
- 适用：综合评估

## 计算公式

```
线性贡献 = 转化值 / 触点数量
时间衰减贡献 = 衰减因子 ^ (n-1) * 转化值
```

## 适用场景

- 渠道效果评估
- 营销活动评估
- 用户旅程优化
