# 异常值校验钩子

## 核心功能

检测数据中的异常值

## 校验规则

### 统计方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 3σ原则 | 超出均值±3σ为异常 | 正态分布数据 |
| IQR法 | 超出Q1-1.5*IQR或Q3+1.5*IQR | 非正态分布 |
| 箱线图 | 识别极端值 | 通用 |

### 业务规则

| 规则 | 阈值 | 处理 |
|------|------|------|
| 金额异常 | <0或>1000000 | 预警 |
| 数量异常 | <0或>10000 | 预警 |
| 比率异常 | <0或>1 | 预警 |

### 阈值标准

| 场景 | 阈值 | 处理方式 |
|------|------|----------|
| 关键指标 | 存在 | 预警 |
| 一般指标 | ≥3% | 预警 |
| 可疑记录 | 自动标记 | 记录日志 |

## 校验内容

1. 数值范围检查
2. 统计异常检测
3. 业务规则异常
4. 异常值分布

## 触发时机

- 取数任务完成后（后置）
- 清洗任务完成后（后置）

## 处理方式

| 场景 | 处理 |
|------|------|
| 关键指标异常 | 预警 |
| 异常值≥3% | 预警 |
| 业务规则异常 | 预警 |
| 异常值可解释 | 记录原因 |

## 输出结果

- 异常值清单
- 异常值统计
- 处理建议
