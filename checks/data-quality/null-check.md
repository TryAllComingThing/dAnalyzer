# 空值校验钩子

## 核心功能

检测数据中的空值情况

## 校验规则

### 空值类型

| 类型 | 说明 |
|------|------|
| NULL | 数据库NULL值 |
| 空字符串 | '' 或 "" |
| 空格 | 仅包含空格 |
| NaN | 非数值 |
| None | Python空值 |

### 阈值标准

| 字段类型 | 阈值 | 处理方式 |
|----------|------|----------|
| 关键字段 | 存在1条 | 终止任务 |
| 主键字段 | 存在1条 | 终止任务 |
| 数值字段 | ≥5% | 预警 |
| 日期字段 | ≥5% | 预警 |
| 字符字段 | ≥10% | 预警 |
| 非关键字段 | ≥30% | 预警 |

### 关键字段识别

- 用户ID：user_id
- 订单ID：order_id
- 商品ID：product_id
- 金额：amount、price
- 时间：create_time、order_time

## 校验内容

1. 各字段空值统计
2. 空值占比计算
3. 关键字段空值检测
4. 空值分布分析

## 触发时机

- 取数任务完成后（后置）
- 清洗任务完成后（后置）

## 处理方式

| 场景 | 处理 |
|------|------|
| 关键字段空值 | 终止任务 |
| 数值字段空值≥5% | 预警 |
| 一般字段空值≥30% | 预警 |
| 空值可填充 | 建议填充值 |

## 输出结果

- 空值统计表（字段名、空值数、空值率）
- 关键字段空值清单
- 处理建议
