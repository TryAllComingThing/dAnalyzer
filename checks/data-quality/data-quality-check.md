# 数据质量校验钩子

## 核心功能

检测数据中的空值、异常值、重复值、断层问题

## 校验规则

### 空值校验

- 数值型字段：空值占比≥5% → 预警
- 关键字段：空值≥1条 → 终止

### 异常值校验

- 3σ原则识别：超出均值±3σ为异常
- 异常值占比≥3% → 预警
- 关键指标异常值 → 预警

### 重复值校验

- 按主键检查重复
- 重复记录自动标记/剔除

### 断层校验

- 时间序列数据检查连续性
- 缺失日期 → 预警

## 触发时机

- 取数任务完成后（后置）
- 清洗任务完成后（后置）

## 处理方式

| 校验项 | 阈值 | 处理方式 |
|--------|------|----------|
| 数值空值 | ≥5% | 预警 |
| 关键字段空值 | ≥1 | 终止 |
| 异常值 | ≥3% | 预警 |
| 重复记录 | 存在 | 自动剔除 |
| 时间断层 | 存在 | 预警 |

## 输出结果

- 校验报告
- 异常数据列表
- 处理建议
