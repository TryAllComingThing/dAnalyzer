# 重复值校验钩子

## 核心功能

检测数据中的重复记录

## 校验规则

### 重复类型

| 类型 | 说明 | 检测方式 |
|------|------|----------|
| 完全重复 | 所有字段值相同 | 全字段GROUP BY |
| 主键重复 | 主键字段值相同 | 主键字段GROUP BY |
| 部分重复 | 关键字段组合重复 | 关键字段组合GROUP BY |

### 关键字段组合

| 数据类型 | 关键字段 |
|----------|----------|
| 订单数据 | order_id 或 order_id+dt |
| 用户数据 | user_id |
| 商品数据 | product_id/sku_id |
| 行为数据 | user_id+behavior_type+dt |

### 阈值标准

| 场景 | 阈值 | 处理方式 |
|------|------|----------|
| 主键重复 | 存在 | 终止任务 |
| 完全重复 | ≥1条 | 自动剔除+预警 |
| 部分重复 | ≥5% | 预警 |

## 校验内容

1. 主键重复检测
2. 完全重复记录检测
3. 部分重复检测
4. 重复比例统计

## 触发时机

- 取数任务完成后（后置）
- 清洗任务完成后（后置）

## 处理方式

| 场景 | 处理 |
|------|------|
| 主键重复 | 终止任务+日志 |
| 完全重复 | 自动剔除+预警 |
| 部分重复 | 预警+建议去重 |
| 无重复 | 通过 |

## 输出结果

- 重复记录统计
- 重复记录明细（前10条）
- 处理建议
