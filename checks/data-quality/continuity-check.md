# 连续性校验钩子

## 核心功能

检测时间序列数据的连续性

## 校验规则

### 时间连续性

| 场景 | 检查内容 | 阈值 |
|------|----------|------|
| 日数据 | 每日数据是否完整 | 缺失>1天预警 |
| 周数据 | 每周数据是否完整 | 缺失>1周预警 |
| 月数据 | 每月数据是否完整 | 缺失>1月预警 |

### 业务连续性

| 场景 | 检查内容 | 阈值 |
|------|----------|------|
| 订单连续性 | 订单ID是否连续 | 断号预警 |
| 用户连续性 | 用户ID是否连续 | 断号预警 |
| 金额连续性 | 金额变化是否合理 | 突变>50%预警 |

### 阈值标准

| 场景 | 阈值 | 处理方式 |
|------|------|----------|
| 时间缺失 | ≥1天 | 预警 |
| 数据中断 | 存在 | 预警 |
| 数值突变 | ≥50% | 预警 |

## 校验内容

1. 时间序列完整性
2. ID序列连续性
3. 数值连续性检查
4. 断层检测

## 触发时机

- 取数任务完成后（后置）
- 日报生成前（前置）

## 处理方式

| 场景 | 处理 |
|------|------|
| 时间缺失 | 预警+标记 |
| 数据断层 | 预警 |
| 数值突变 | 预警+原因记录 |

## 输出结果

- 连续性检查报告
- 缺失数据清单
- 处理建议
