# 数据类型校验钩子

## 核心功能

检测字段数据类型是否符合预期

## 校验规则

### 类型定义

| 期望类型 | 正确格式 | 示例 |
|----------|----------|------|
| 整数 | 数字，无小数 | 100, -5 |
| 浮点数 | 数字，可有小数 | 100.5, -3.14 |
| 字符串 | 任意字符 | abc, 中文 |
| 日期 | YYYY-MM-DD | 2026-04-24 |
| 日期时间 | YYYY-MM-DD HH:MM:SS | 2026-04-24 10:30:00 |
| 布尔值 | true/false, 0/1 | true, false |
| 手机号 | 11位数字 | 13812345678 |
| 邮箱 | xxx@xxx.xx | test@example.com |

### 常见问题

| 问题类型 | 说明 | 检测方式 |
|----------|------|----------|
| 类型不匹配 | 字符串类型填入数值 | 正则匹配 |
| 格式错误 | 格式不符合规范 | 格式验证 |
| 隐式转换 | 数据库自动转换 | 样例检测 |

## 校验内容

1. 字段类型检查
2. 格式规范检查
3. 类型兼容性检查

## 触发时机

- 取数任务完成后（后置）
- 数据清洗后（后置）

## 处理方式

| 场景 | 处理 |
|------|------|
| 类型不匹配 | 预警+记录 |
| 格式错误 | 预警+建议 |
| 批量错误 | 终止+日志 |

## 输出结果

- 类型检查报告
- 错误字段清单
- 修复建议
