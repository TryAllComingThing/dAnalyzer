# 计算逻辑校验钩子

## 核心功能

校验分析任务中的计算逻辑是否符合规范

## 校验内容

### 1. 公式校验

- 指标计算公式是否正确
- SQL聚合逻辑是否规范
- 除零处理是否合理

### 2. 聚合方式校验

- SUM/AVG/COUNT使用场景
- 去重逻辑（COUNT DISTINCT）
- 时间窗口函数规范

### 3. 数值精度校验

- 保留小数位数
- 四舍五入规则
- 百分比计算精度

### 4. 边界处理校验

- 空值参与运算的处理
- NULL vs 0 的区分
- 日期边界的处理

## 校验规则

| 校验项 | 规则 | 处理方式 |
|--------|------|----------|
| 公式错误 | 计算逻辑不合法 | 终止任务 |
| 聚合滥用 | 不必要的COUNT DISTINCT | 预警 |
| 精度丢失 | 小数位过多/过少 | 预警 |
| 空值处理不当 | 未处理NULL直接运算 | 终止任务 |

## 触发时机

- CLI前置校验：任务执行前

## 输出结果

- 校验报告
- 问题项列表
- 修正建议
