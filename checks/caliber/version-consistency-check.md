# 版本一致性校验钩子

## 核心功能

校验数据版本与资产版本的一致性

## 校验规则

### 资产版本检查

| 资产类型 | 检查内容 | 处理方式 |
|----------|----------|----------|
| 指标字典 | 计算逻辑是否最新 | 预警 |
| SQL模板 | 模板是否最新 | 预警 |
| 分析模型 | 模型版本 | 预警 |
| 报告模板 | 模板版本 | 预警 |

### 数据版本检查

| 检查项 | 说明 | 处理 |
|--------|------|------|
| 指标版本 | 使用指标字典版本 | 预警 |
| 计算版本 | 计算逻辑版本 | 预警 |
| 输出版本 | 输出格式版本 | 预警 |

### 版本规则

- 资产版本格式：v1.0、v1.1
- 数据版本格式：20260424
- 配置文件格式：JSON/YAML

## 校验内容

1. 资产版本检查
2. 计算逻辑版本
3. 输出格式版本

## 触发时机

- 取数任务执行前（前置）
- 报告生成前（前置）

## 处理方式

| 场景 | 处理 |
|------|------|
| 使用旧版本 | 预警+建议更新 |
| 版本缺失 | 预警 |
| 版本冲突 | 终止+日志 |

## 输出结果

- 版本检查报告
- 版本差异说明
- 更新建议
